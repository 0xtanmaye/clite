# Command-Line Text Editor (CLiTE)

A minimalist command-line text editor built from scratch in ~1300 lines of C++, implemented in a single file with no external dependencies.  
It supports essential text editing features, syntax highlighting, and search functionality.

This project was created for learning purposes and was inspired by the *Kilo* text editor tutorial:
https://viewsourcecode.org/snaptoken/kilo/

## Features
- Basic text editing
- Syntax highlighting
- Incremental search
- Single-file implementation
- No external dependencies
